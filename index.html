HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body is considered a parent element to its inside eleements (children)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When building responsive designs, the most important tools are flex-grow, flex-shrink, justify-content, and the align-item properties.
  • justify-content aligns flex items along the main axis and align-item aligns flex items along the cross axis.
  • You can also use the flex property to assign flex-grow, flex-shrink, and basis properties in a shorthand version.
  • This is an example: flex: flex-grow, flex-shrink, basis

GIT

  • git status: checks what branch you are on
  • git checkout "name of branch": takes you to that branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin branch-name: pulling most up to date respective branch from cloud

JAVASCRIPT

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Javascript reads code from top-to-bottom
  • Arrays in Javascript can store many different data types
  • A function is Javascript allows us to re-use codes of block that perform specific tasks
  • You call a function by writing the name of the function with parentheses(eg. thisIsAFunction();)
